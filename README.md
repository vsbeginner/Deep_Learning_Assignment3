# Deep_Learning_Assignment3
This project implements a Text Sentiment Classification system using deep learning models such as Recurrent Neural Networks (RNN) and Long Short-Term Memory (LSTM). The system is designed to analyze textual data like reviews, comments, or messages and classify them into different sentiment categories such as positive, negative, or neutral.

The project begins with preprocessing the text data, which includes cleaning, tokenization, and converting words into numerical representations that can be understood by machine learning models. After preprocessing, both RNN and LSTM models are built and trained on the dataset to learn patterns in the text.

RNN helps in processing sequential data, while LSTM improves performance by handling long-term dependencies and avoiding issues like vanishing gradients. The models are then evaluated based on their accuracy and performance in predicting sentiments.

This project demonstrates how deep learning techniques can be effectively applied in natural language processing tasks, and highlights the advantages of LSTM over traditional RNN models in sentiment analysis.
